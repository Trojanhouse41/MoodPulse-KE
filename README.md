📋 What It Does
MoodPulse-KE is an interactive wellness companion designed for Kenyan students. It helps users:
-  Track daily mood and stress levels
-  Access localized mental health resources
-  Get study-wellness tips tailored to the Kenyan academic context
-  Connect with peer support strategies

 Features
-  Built for Kenyan students, by a Kenyan developer
-  Interactive, conversational UI powered by PartyRock (AWS)
-  Private, judgment-free space for wellness check-ins
-  Works on any device with a browser
